Sistema de Pré-Matrículas

- Sobre o projeto

O Sistema de Pré-Matrículas foi desenvolvido com o objetivo de facilitar o processo de inscrição de alunos na rede municipal de ensino, 
oferecendo mais praticidade para pais e responsáveis e maior organização para a gestão escolar.
O sistema permite realizar o cadastro de alunos, associando-os às escolas mais próximas de acordo com o bairro informado, 
respeitando critérios como faixa etária e capacidade das turmas. Além disso, conta com um painel administrativo para acompanhamento e gerenciamento das pré-matrículas realizadas.
Este projeto foi desenvolvido com fins educacionais e demonstrativos, priorizando simplicidade e clareza na implementação.

- Funcionalidades

Cadastro de alunos para pré-matrícula
Associação automática de alunos às escolas conforme o bairro
Verificação de faixa etária e capacidade das turmas
Painel administrativo para gerenciamento das pré-matrículas
Listagem e controle de alunos cadastrados

- Tecnologias utilizadas

PHP
MySQL / MariaDB
HTML e CSS
phpMyAdmin (para gerenciamento do banco de dados)

- Banco de dados

O banco de dados será disponibilizado dentro do próprio repositório, contendo a estrutura e os dados necessários para executar o sistema localmente.

Para utilizar:

Crie um banco de dados no MySQL/MariaDB
Importe o arquivo .sql disponibilizado no repositório
Ajuste as credenciais de conexão com o banco no arquivo de configuração do projeto

- Acesso ao painel administrativo

Para acessar o painel administrativo, utilize as seguintes credenciais de exemplo:
E-mail: admin@email.com
Senha: admin

- Atenção:

O método de conferência/autenticação não é seguro e foi implementado apenas como um exemplo didático, 
com o objetivo de manter o sistema mais simples e fácil de compreender. Não é recomendado utilizar este modelo de autenticação em ambientes de produção.

- Como executar o projeto

Clone este repositório:

git clone https://github.com/Ithyel-Cavalcante/sistema-prematriculas.git

Coloque o projeto em um servidor local (XAMPP, WAMP ou Laragon)
Importe o banco de dados conforme instruções acima
Acesse o sistema pelo navegador
